# Sistema de Gerenciamento de Alunos em Python

## Objetivo do Projeto
Este projeto foi desenvolvido com o objetivo de praticar lógica de programação utilizando Python puro, através da construção de um sistema simples de gerenciamento de alunos em terminal.

O sistema permite cadastrar, visualizar, buscar, remover alunos e calcular a média geral das notas armazenadas.

---

## Funcionalidades
- Cadastro de alunos com nome, idade e nota
- Listagem de todos os alunos cadastrados
- Busca de aluno pelo nome
- Remoção de aluno da lista
- Cálculo da média geral das notas
- Validação de nota entre 0 e 10
- Tratamento básico de entradas inválidas

---

## Tecnologias Usadas
- Python 3

---

- Estruturas condicionais (`if`, `elif`, `match case`)
